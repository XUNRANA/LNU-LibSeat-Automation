# 📦 Release Notes — LNU-LibSeat v3.0.0

## ✨ 新特性

- 🖥️ **全新 GUI**：CustomTkinter Indigo 主题，卡片式布局
- 🧵 **多线程并发**：多账号同时运行，分时段分工
- 🎯 **全自习室地毯式扫描**：首选 → 全量随机扫描
- 🔐 **验证码双引擎**：图鉴 API + ddddocr 本地兜底
- ⏱️ **精确卡点**：提前 30s → 提前 6s 锁定 → 整点提交
- 🎥 **浏览器全程录屏**：MP4 格式
- 📁 **会话级追溯**：日志 + 截图 + 录屏 + 抢座顺序
- 📧 **邮件通知**：抢座成功自动发送战报
- 🛡️ **黑名单检测**：被拉黑自动停止，避免无效请求
- 📦 **EXE 免安装分发**

## ⚠️ 已知限制

- 高峰期（06:30）竞争极其激烈，即使验证码全部通过，座位仍可能被更快用户抢走
- 图鉴 API 在高并发时延迟波动大（3.5s ~ 17.8s）
- 本地 OCR 约 38.8% 概率需要重试，每次重试约浪费 1~2 秒
