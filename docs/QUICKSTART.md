# ⚡ 快速上手教程

本教程将带你从零开始配置并运行 LNU-LibSeat-Automation，完成自动化抢座。

---

## 📋 小白必看：电脑环境要求

如果这是一台**全新的电脑**，你**只需要**满足以下 3 点，没有任何其他“隐藏要求”：

1. **💻 操作系统**：Windows 10 或 Windows 11。
2. **🌐 浏览器环境**：电脑里已经可以正常使用 **Microsoft Edge** 或者 **Google Chrome**（二选一即可）。必须是相对较新的版本。
3. **📶 网络环境**：电脑当前连接的网络能够正常打开“辽宁大学座位预约系统”。

🎯 **你绝对不需要准备的东西**（程序全部帮你打包好了）：
- ❌ **不需要** 下载安装 Python，也不需要配置环境变量。
- ❌ **不需要** 接触命令行黑窗、不用装代码编辑器。
- ❌ **不需要** 自己去搜教程下载安装浏览器驱动 (WebDriver)，程序会自动在后台帮你匹配并下发！

## 方式一：下载 EXE 直接运行（推荐零基础用户）

> 完全不需要安装 Python，下载即用！

### Step 1：下载程序

前往 [GitHub Releases](https://github.com/XUNRANA/LNU-LibSeat-Automation/releases/latest) 页面，下载最新版 `LNU-LibSeat-v2.8.0.zip`，解压到任意位置。

### Step 2：运行

双击 `LNU-LibSeat.exe`，程序会打开图形界面（窗口 `1400x980`，启动时居中显示）。

### Step 3：配置

在 GUI 中填写以下内容：

1. **目标设置**：选择校区、自习室，填写最多 10 个优先座位号（按优先级排列）
2. **账号信息**：填写学号和密码；如需双账号分时段，打开「启用副账号」开关
3. **成功通知**：填写接收邮箱（可选）
4. **执行模式**：选择「立即执行」或「定时执行」

### Step 4：开始抢座

点击「🚀 开始抢座」按钮！程序会在右侧日志区域实时显示运行状态。

## 方式二：Python 源码运行（开发者 / 高级用户）

### Step 1：克隆并安装环境

```powershell
# 克隆项目 (或者直接在 GitHub 页面点击 Download ZIP 解压)
git clone https://github.com/XUNRANA/LNU-LibSeat-Automation
cd LNU-LibSeat-Automation
```

> 💡 **环境免配置提示**：你**不需要**手动输入繁琐的命令去安装依赖或虚拟环境！
> 只要你电脑里安装了 Python，双击运行 `run.bat` 时，系统会自动在后台帮你构建隔离环境并下载所有必须的依赖包。

### Step 2：启动 GUI

直接双击 `run.bat`，或在终端中运行：

```powershell
python gui.py
```

GUI 会自动打开，在界面中完成配置后点击「开始抢座」即可。

> 也可以手动编辑 `config.py` 进行高级配置，然后运行 `python main.py` 直接执行。

## 🔧 常见问题 FAQ

### Q: 浏览器报错 "driver not found" 怎么解决？
**A**: 确保网络畅通，内置的 `webdriver-manager` 会自动下载匹配你浏览器版本的驱动。如果下载失败，可手动下载 WebDriver 并在 `config.py` 里配置 `DRIVER_PATH = "驱动完整路径.exe"`。

### Q: 验证码长期识别失败怎么办？
**A**: v2.8.0 已集成图鉴 (TTShiTu) 付费 API 作为优先识别引擎，ddddocr 本地引擎作为兜底。点选验证码支持 **1~4 字按顺序点击**，每个座位最多 10 次验证码重试机会。

### Q: 抢座失败了怎么排查？
**A**: 查看 `logs/` 文件夹：
- `lnu_seat.log` — 完整运行日志
- `screenshot_*.png` — 失败时的页面截图
- `recordings/*.mp4` — 浏览器全程录屏视频（v2.8.0 新增）

### Q: 如果我提前很久点「开始抢座」，电脑中途休眠了怎么办？
**A**: V2.3.0 引入了**完全底层的防休眠机制**。只要你不关闭软件，点击开始抢座后，程序会自动向 Windows 申请系统唤醒权限，**电脑绝对不会进入睡眠状态，也不会断网！**你可以放心挂机 10 小时以上。

### Q: 怎么做到每天自动开机运行完全自动化（不用每天手动运行）？
**A**: 结合 Windows 的【任务计划程序】：
1. 新建任务，触发器设为每天 `00:15`（凌晨，程序会自动等到你设置的目标时刻，默认 6:30）
2. 操作选"启动程序"，程序选 `LNU-LibSeat.exe`（EXE 模式）或 `run.bat`（Python 模式）
3. 起始于（Start in）填入程序所在的**完整文件夹路径**
4. 勾选"唤醒计算机运行此任务"
